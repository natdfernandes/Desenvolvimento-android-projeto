# Plano

Neste documento será registrado so prompts utilizados para a contrução do app.


## Tela de login

Crie uma tela de login. Para essa tela:
- cada item do container deve ter 24px de espaçamento (gap), padding de 24px horizontal e 16px vertical
- Header container
    - inicialmente insira a imagem presente em assets\images\logo.png
    - Abaixo insira o header com valor "BEM VINDO"
    - Abaixo o sub header com valor "SEU APP FAVORITO DE COMPRAS"
- Formulario container
    - deve ter um gap de 8px
    - insira um input de email, com placeholder "Email"
    - insira um input de senha, com placeholder "Senha"
    - todos os inputs deve ter border radius de 9999, gap de 4px, borda de 1px na cor #D4D4D4 e texto na cor #A3A3A3, padding 20px horizontal e 8px vertical
- Controle de acesso container
    - deve ter um gap de 8px
    - inserir um link com texto "Esqueceu sua senha?", na cor #0284C7
    - inserir um checkbox com label "Lembrar senha", na cor #171717. O checkbox deve ter borda de 2px na cor #A3A3A3 e border radius de 2px
- Botão entrar
    - deve ter width 100%
    - Texto "ENTRAR"
    - cor #0891B2 e texto na cor #FAFAFA
    - gap de 4px e padding de 12px
    - border radius 4px
- Footer
    - deve ter width 100%
    - paragrafo com texto "Desenvolvido por: Natalia Dias Fernandes 2026 ®"
    - cor de texto #0284C7
